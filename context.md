# JICC 開發上下文文檔

## 項目概述
JICC 是一個企業資源規劃系統，旨在為中小企業提供完整的業務管理解決方案。

## 當前狀態
- 已有基礎的 FastAPI 後端框架
- 使用 SQLite 數據庫
- 包含基本的用戶認證和交易管理功能
- 簡單的 HTML 前端界面

## 目標架構

### 後端技術棧
- **框架**: FastAPI (Python 3.9+)
- **ORM**: SQLAlchemy
- **數據庫**: PostgreSQL (生產環境) / SQLite (開發環境)
- **認證**: JWT + OAuth2
- **緩存**: Redis
- **任務隊列**: Celery
- **文檔**: Swagger/OpenAPI

### 前端技術棧
- **框架**: React 18 + TypeScript
- **UI 庫**: Material-UI (MUI) 或 Ant Design
- **狀態管理**: Redux Toolkit + RTK Query
- **路由**: React Router v6
- **表單**: React Hook Form + Yup
- **圖表**: Chart.js 或 Recharts

### 數據庫設計

#### 核心實體
1. **用戶管理**
   - users (用戶表)
   - roles (角色表)
   - permissions (權限表)
   - user_roles (用戶角色關聯)

2. **財務管理**
   - accounts (會計科目)
   - transactions (交易記錄)
   - invoices (發票)
   - payments (付款記錄)

3. **庫存管理**
   - products (產品表)
   - categories (產品分類)
   - inventory (庫存記錄)
   - suppliers (供應商)

4. **客戶關係**
   - customers (客戶表)
   - contacts (聯繫人)
   - orders (訂單)
   - quotations (報價單)

5. **系統管理**
   - companies (公司信息)
   - settings (系統設置)
   - audit_logs (審計日誌)

## API 設計原則

### RESTful API 規範
- 使用標準 HTTP 方法 (GET, POST, PUT, DELETE)
- 統一的響應格式
- 適當的 HTTP 狀態碼
- 分頁和過濾支持
- API 版本控制

### 響應格式

```json
{

  "errors": [],
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 100,
    "pages": 5
  }
}
```
